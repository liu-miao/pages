知识星球Introduction

Simplicity First

以 Markdown 为中心的项目结构，以最少的配置专注于写作。

Vue-Powered

享受 Vue 的开发体验，可以在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。

Performant

VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。

Themes

提供了一个开箱即用的默认主题。你也可以选择社区主题或创建自己的主题。

Plugins

灵活的插件 API，使得插件可以为你的网站提供许多即插即用的功能。

Bundlers

默认的打包工具是 Vite，同时也支持 Webpack。选择你喜欢的工具！
